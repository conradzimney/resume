LAMP Server

Web server and client site that shows movie box office revenues from 2014, allowing the user to move between a list view and a detail view. On the list view the user can search for a given movie by title, and on the detail view the user can view information from an Open Movie Database API (IMDB and Rotten Tomatoes APIs). Uses the LAMP Stack: Linux, Apache, MySQL and PHP, hosted on a Digital Ocean Droplet.

You can find the source code on my GitHub here!

REST API Node Server

Authenticates Users and Manages Profiles with the MEAN stack. Uses the express framework to build a basic Node.js server. Uses the express static middleware to serve static files. Uses the passport module to provide user user authentication using multiple strategies. Persists user information and session state in a shared data store. Provides a simple REST API that the client pages calls via AJAX. Designed and implemented a proper RESTful API for a given set of resources. Enforces data validation and authorization rules on the server. Wrote automated tests using Mocha that verifies that the server code and API still work after changes are made.

You can find the source code on my GitHub here!

Go Suggestion Service

For this project I have built a suggestion service that returns possible matches based on what a user has typed so far. These kinds of services need to be very high-performance, so I have used the Go programming language to build a compiled executable, and have implementd a trie data structure to make finding matches very efficient.
